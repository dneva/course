Для начала работы с программой запустите файл CRM.EXE. Для работы программы файлы из папки не должны изменяться или удаляться. 
Программа предназначена для работы с записями о клиентах. Функции, включенные в проект: добавление/удаление записи, поиск и сортировка записей.
Для работы с программой рекомендуется ознакомиться с Руководством пользователя, входящим в документацию к данной системе.
